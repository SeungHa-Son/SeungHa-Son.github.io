## 📁 Repository Structure (Key Files)
- `index.md`  
  메인 페이지 (이력 요약 + Skills)

- `skills.md`  
  기술 스택을 문서 형태로 정리한 페이지

- `activities.md / awards.md / education.md`  
  메뉴에서 접근 가능한 세부 이력 페이지

- `_config.yml`  
  사이트 전역 설정 및 테마 구성 파일

- `assets/`  
  이미지, 아이콘, 커스텀 스타일 등 정적 리소스
